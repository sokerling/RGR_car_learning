ML & Big Data Dashboard for Model Inference

Краткое описаниеЭтот проект представляет собой веб-дашборд на Streamlit для инференса (вывода) нескольких моделей машинного обучения и анализа данных. На одной платформе можно:

Загружать данные о недвижимости (CSV) или вводить их вручную

Одновременнo запускать шесть моделей (дерево решений, градиентный бустинг, CatBoost, бэггинг, стэкинг, полносвязная нейросеть)

Видеть табличные результаты и гистограммы распределения предсказаний

Изучать визуализации исходного датасета: гистограммы, scatter-плоты, boxplot и тепловую карту корреляций
